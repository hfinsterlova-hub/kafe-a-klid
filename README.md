# Kafe a klid

Interaktivní prototyp aplikace pro kavárnu podle zadání Czechitas a BPMN procesu objednávky.

## Co prototyp umí

- zákaznickou objednávku z aktuální nabídky,
- varianty nápojů: velikost, mléko a cukr,
- simulaci platby a kontrolu skladu,
- frontu pro obsluhu se stavy objednávky,
- notifikaci, že je objednávka hotová,
- skladový pohled na zásoby a dostupnost položek,
- admin pohled na ceník, nabídku a provozní statistiky.

## Lokální spuštění

```bash
npm install
npm run dev
```

## Ověření

```bash
npm test
npm run build
```
